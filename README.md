# AI-Powered Financial Goal Planning and Advisory System

## Overview

AI-Powered Financial Goal Planning and Advisory System is a web application that helps users analyze their financial health, plan future goals, and receive intelligent financial recommendations.

The system uses Machine Learning to predict whether a financial goal is achievable based on income, expenses, savings, EMIs, and target goals.

## Features

* Financial Health Score Calculation
* Expense Analysis Dashboard
* Monthly Savings Tracking
* Goal Planning (House, Car, Travel)
* Machine Learning Based Goal Prediction
* Personalized Financial Recommendations
* Interactive Charts and Visualizations

## Project Workflow

1. User enters financial information.
2. System calculates expenses and savings.
3. Machine Learning model evaluates goal feasibility.
4. Financial health score is generated.
5. Personalized recommendations are provided.
